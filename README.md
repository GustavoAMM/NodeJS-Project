# NodeJS project

Instalar el proyecto:
```
npm install
```

Iniciar proyecto:
```
npm run dev
```
Comando completo para iniciar el proyecto:
```
nodemon src/index.js
```

Link del proyecto => [Angel-NodeJS](https://angel-nodejs.herokuapp.com/)
